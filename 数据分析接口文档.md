# 数据分析接口文档

> 版本：v1.0，2026-07-21
> 适用对象：前端组
> 基础路径：`/api/analysis`

---

## 1. 通用说明

- 所有接口返回统一格式 `{ "code": 200, "message": "success", "data": {...} }`
- 请求需携带 Token：`Authorization: Bearer {token}`
- 当前所有接口数据来源为算法组 `steel_data_workspace.mat`，暂为静态数据

---

## 2. 变量相关性矩阵

### 2.1 接口

```
GET /api/analysis/correlation
```

### 2.2 说明

获取 6 个能源变量两两之间的 Pearson 相关系数矩阵。数据基于 2018 全年 35,040 个 15 分钟粒度样本计算，由算法组提供（`steel_data_workspace.mat / corr_mat`）。

矩阵行列顺序一致，均为：

| 序号 | 英文字段（label） | 中文名 | 单位 |
|---|---|---|---|
| 0 | `elec` | 用电量 | kWh |
| 1 | `Lagging_Current_Reactive_Power_kVarh` | 滞后无功电量 | kVarh |
| 2 | `Leading_Current_Reactive_Power_kVarh` | 超前无功电量 | kVarh |
| 3 | `CO2_tCO2_` | 碳排放 | tCO2 |
| 4 | `Lagging_Current_Power_Factor` | 滞后功率因数 | % |
| 5 | `Leading_Current_Power_Factor` | 超前功率因数 | % |

### 2.4 字段说明

| 字段 | 类型 | 说明 |
|---|---|---|
| `labels` | `string[]` | 变量英文字段名，长度 6 |
| `labelsCn` | `string[]` | 变量中文名，与 labels 一一对应 |
| `units` | `string[]` | 变量单位，与 labels 一一对应 |
| `matrix` | `number[][]` | 6×6 相关系数矩阵，`matrix[row][col]` 为 labels[row] 与 labels[col] 的 Pearson 相关系数。对称矩阵，对角线 = 1.0 |
| `source` | `string` | 数据来源说明 |
| `sampleCount` | `int` | 样本量：35040 |
| `timeRange` | `string` | 数据时间范围 |

### 2.5 关键发现（供前端展示标注）

- **用电量与碳排放高度正相关**（r = 0.988），几乎是线性绑定关系
- **用电量与滞后无功电量强正相关**（r = 0.896），高负荷时段无功消耗同步上升
- **超前无功电量与超前功率因数强负相关**（r = -0.944），超前无功越大超前功率因数越低
- 滞后/超前功率因数整体相关性较弱，说明两者反映不同的电网工况

### 2.6 前端渲染建议

热力图颜色映射：
- **正相关（> 0）** → 暖色渐变（浅橙 → 深红），r = 1.0 为最深红
- **负相关（< 0）** → 冷色渐变（浅蓝 → 深蓝），r = -1.0 为最深蓝
- **接近 0** → 白色或浅灰
- **对角线（r = 1.0）** → 可用深色或特殊标记区分
